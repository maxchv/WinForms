Написать приложение для автосолона.

Для каждого автомобиля представить данные:
	int Id
	string Model
	decimal Price
	bool Sold

Приложение состоит из двух окно.
В основном окне присутствует список авто (ListBox) и кнопка Добавить.
При нажатии нак нопку Добавить отображается немодальное диалоговое окно
в котором есть возможность создать или отредактировать данные автомобиля.
В дочернем окне поместить кнопки: Добавить, Обновить, Удалить.

Обеспечить взаимодействие между основным и дочерним окном:
	1. При добавлении автомобиля в дочернем окне (нажатие кнопки Добавить)
	   приводит к появлению новой записи в списке автмомобилей
	2. Двойной клик по записи в списке автомобилей приводит к загрузке данных
           о выбранном автомобиле в дочернее окно
	3. Обновление данных автомобиля в дочернем окне (нажатие кнопки Обновить) приводит к
           обновлении записи в списке автомоблией основного окна
	4. Удаление автомобиля (нажатие кнопки Удалить) в дочернем окне также приводит к удалению
	   соответсвующей записи в списке автомобилей основоного окна

Дополнительное задание:
	1. Сохранять/загружать данные в xml
	2. Удалять запись при нажатии кнопки Delete на клавиатуре в списке ListBox
