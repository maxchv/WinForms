 Создать приложение по учету сотрудников организации 
 В качестве информации о сотруднике должны присутствовать

•	ФИО;
•	Должность;
•	Статус;
•	Уровень доступа;
•	Сетевое имя;
•	Дата рождения;
•	Дата приема на работу.

Интерфейс должен предоставлять менеджеру по персоналу
•	Возможность найти информацию о сотруднике по любым данным;
•	Редактировать данные о сотруднике;
•	Добавлять нового сотрудника.

Список всех сотрудников должен находится в ListBox.
Данные каждого сотрудника закгружаются в TabCotrol в виде отдельного таба.
Реализовать функции управления табами через контекстое меню (закрыть текущую вкладку,
все вкладки, все, кроме текущей)

Реализовать функцию сохранения/считывания данных сотрудников (xml)
