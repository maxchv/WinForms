Написать приложение для поиска файлов и папок на компьютере.
Для ускорения поиска реализовать функцию индексации данных.
Индексация запускается при клике по кнопку "Запустить индексацию",
при этом выполняется сканирование всех файлов и папок на
всех дисках на компютере с записью полного пути к файлам и папкам в
индексный  текстовый файл.
Индексация выполняется для каждого диска в отдельном потоке через
BackgroundWorker.
Предусмотреть возможность остановки индексации файлов.
По завершении индексации на всех дисках вывести соответсвующее
сообщение в label или статусную строку.
Поиск файлов выполнять по содержимому индексных файлов.
